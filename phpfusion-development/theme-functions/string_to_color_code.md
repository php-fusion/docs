# string\_to\_color\_code\(\)

Versions: `9`

string\_to\_color\_code\( string $text \) : string

### Parameters <a id="parameters"></a>

$text \(string\) \(Required\) Any string.

### Return Values

\(string\) HEX color code.

### Examples

```php
echo string_to_color_code('String');
// a33297
```
