# fusion\_sort\_table\(\)

Versions: `9`

fusion\_sort\_table\( string $table\_id \) : void

### Parameters <a id="parameters"></a>

$table\_id \(string\) \(Required\) The ID of the table.

### Return Values

No value is returned.

### **Examples**

```php
fusion_sort_table('test_table');
echo '<table id="test_table">...</table>';
```
