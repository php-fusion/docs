# showlogo\(\)

Versions: `9`

showlogo\( \[ string $class \] \) : string

### Parameters <a id="parameters"></a>

$class \(string\) \(Optional\) CSS class. Default value: logo

### Return Values

\(string\)

### Examples

```php
echo showlogo();
// <div class='logo'><a href='home.php' title='PHP-Fusion 9'><img src='images/php-fusion-logo.png' alt='Logo'></a></div>
```
