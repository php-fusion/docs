# openside\(\)

Versions: `9`

openside \( string $title \[, string $class \] \) : string

### Parameters <a id="parameters"></a>

$title \(string\) \(Required\) Panel title.

$class \(string\) \(optional\) CSS class. Default value: ''

### Return Values

\(string\)

### **Examples**

```php
openside('Title');
// Content goes here
closeside();
```
