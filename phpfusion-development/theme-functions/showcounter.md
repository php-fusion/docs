description: >-
  If the visitor counter is enabled in settings this function will return the
  number of visitors.
---

# showcounter\(\)

Versions: `9`

showcounter\( void \) : string

### Parameters <a id="parameters"></a>

No parameters.

### Return Values

\(string\)

### **Examples**

```php
echo showcounter();
// 25 unique visits
```
