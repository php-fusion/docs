# fusion\_get\_username\(\)

Versions: `9`

fusion\_get\_username\( int $user\_id \) : string

### Parameters <a id="parameters"></a>

$user\_id \(int\) \(Required\) User ID.

### Return Values

\(string\) Username.

### **Examples**

```php
echo fusion_get_username(1);
// admin
```
