# redirect\_img\_dir\(\)

Versions: `9`

redirect\_img\_dir\( string $source, string $target \) : void

## Parameters <a id="parameters"></a>

$source \(string\) \(Required\) Source path.

$target \(string\) \(Required\) Target path.

## Return Values

No value is returned.

## Examples

```php
redirect_img_dir('path/', 'new/path/');
```
