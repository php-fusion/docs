# random\_string\(\)

Versions: `9`

random\_string\( \[ int $length \] \) : string

## Parameters <a id="parameters"></a>

$length \(int\) \(Optional\) The length of the string. Default value: 6

## Return Values

\(string\) Generated string.

## Examples

```php
echo random_string(10);
// oghHY2Zbag
```
