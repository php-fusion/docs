# random\_string\(\)

Versions: `9`

random\_string\( \[ int $length \] \)

### Parameters <a id="parameters"></a>

$length \(int\) \(Optional\) The length of the string

### Return <a id="return"></a>

\(string\) Generated string

### **Example**

```php
echo random_string(10);
// oghHY2Zbag
```
