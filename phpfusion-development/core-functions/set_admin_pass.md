# set\_admin\_pass\(\)

Versions: `9`

set\_admin\_pass\(string $password\)

### Parameters <a id="parameters"></a>

$password \(string\) \(Required\) Any password

### Return <a id="return"></a>

\(bool\)

### **Example**

```php
set_admin_pass('SECRET_PASSWORD');
```
