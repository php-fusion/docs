# define\_site\_language\(\)

Versions: `9`

define\_site\_language\( string $lang \) : void

### Parameters <a id="parameters"></a>

$lang \(string\) \(Required\) The name of the language.

### Return Values

No value is returned.

### **Examples**

```php
define_site_language('Slovak');
```
