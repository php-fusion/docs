# rrmdir\(\)

Versions: `9`

rrmdir\( string $dir \) : void

### Parameters <a id="parameters"></a>

$dir \(string\) \(Required\) Path to the folder.

### Return Values

No value is returned.

### Examples

```php
rrmdir(IMAGES.'articles');
```
