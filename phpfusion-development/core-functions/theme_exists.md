# theme\_exists\(\)

Versions: `9`

theme\_exists\(string $theme\)

### Parameters <a id="parameters"></a>

$theme \(string\) \(Required\) The theme folder you want to check

### Return <a id="return"></a>

\(bool\)

### **Example**

```php
if (theme_exists('Magazine') {
    //
}
```
