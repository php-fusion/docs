# getusergroups\(\)

Versions: `9`

getusergroups\( void \) : array

### Parameters <a id="parameters"></a>

No parameters.

### Return Values

\(array\) Array of all access levels and user groups.

### Examples

```php
foreach (getusergroups() as $group) {
    //
}
```
