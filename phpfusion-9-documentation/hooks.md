# Hooks

`(Version 7)` `(Version 8)` `(Version 9)`

### Description

Hooks are implementation of bundled action and runtime in specific files to extend specific areas of the system.

| Namespace | Description |
| :--- | :--- |
| **fusion\_login\_connect** | Login actions runtime |
| **fusion\_user\_notices** | Aggregates all user notices |
| **fusion\_adminbar\_pages** | Adds administration menu page |
| **fusion\_user\_pages** | Adds user profile page link to dropdown |
| **fusion\_acp\_action** | Post actions |
| **fusion\_boiler\_paths** | Outputs current boilerplate folder path |
| **fusion\_profile\_page** | Outputs custom profile page |
